# Контрольная работа 3

Предметная область: образовательная платформа.

Реализована база данных в 3НФ со следующими таблицами:
- authors
- courses
- lessons

Что сделано:
- первичные ключи во всех таблицах
- внешние ключи
- индексы по внешним ключам и часто используемым полям
- полнотекстовый поиск по полям `title` и `description`
- используется словарь русского языка `russian`
- частичный поиск реализован с помощью `pg_trgm`

Для проверки производительности использован `EXPLAIN (ANALYZE, BUFFERS)`.

Полнотекстовый поиск использует индекс `idx_lessons_fts`.
Частичный поиск использует индексы `idx_lessons_title_trgm` и `idx_lessons_description_trgm`.
